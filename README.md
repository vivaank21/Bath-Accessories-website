# 🛁 Jigna Bathroom Accessories – Premium Bathroom Collection Website

A modern, fully animated, luxury bathroom products landing page built with **HTML, CSS, and JavaScript**, featuring smooth animations powered by **GSAP** and **ScrollTrigger**.

This project showcases a premium brand experience with parallax effects, scroll animations, 3D tilt interactions, and polished UI transitions.

---

## ✨ Features

- 🎨 Elegant, luxury UI design
- 🚀 Smooth scroll-based animations (GSAP + ScrollTrigger)
- 🌊 Parallax scrolling effects
- 🧩 3D tilt interaction on collection cards
- 🛒 Animated "Add to Cart" button effects
- 📊 Animated stats counters
- 📩 Contact form animation
- 📬 Newsletter subscription interaction
- 🖱 Custom cursor (desktop only)
- ⚡ Performance optimizations
- 📱 Fully responsive design

---

## 🛠 Tech Stack

- **HTML5**
- **CSS3 (Custom Properties / Variables)**
- **Vanilla JavaScript**
- **GSAP (GreenSock Animation Platform)**
- **ScrollTrigger Plugin**
- **Intersection Observer API**

## 🚀 How to Run Locally

1. Download or clone the repository:
   ```bash
   git clone https://github.com/vivaank21/Bath-Accessories-website.git
   ```

2. Navigate into the project folder:
   ```bash
   cd Bath-Accessories-website
   ```

3. Open `index.html` in your browser.

No build tools or installations required.

## 🎨 Customization Guide

### Change Brand Colors
Modify CSS variables inside `styles.css`:

```css
:root {
  --primary-sage: #8b9d83;
  --accent-terracotta: #c87f5b;
  --dark-charcoal: #2c2c2c;
}
```

### Update Content
- Edit text and sections inside `index.html`
- Replace product images
- Update logo (`logo.png`)

### Modify Animations
All animations are handled inside `script.js`.  
You can adjust:

- Duration
- Easing
- Stagger timing
- Scroll trigger positions
